TAGI

  • "/br" - nowa linia

  • "strong" - podkreśla wysoką istotność i poważność frazy w nim zawartej

  • "em" - podkreśla, że tekst może mieć inne znaczenie (np. ironia)

  • "b" - pogrubienie (ale nie ma znaczenia semantycznego)

  • "i" - kursywa (ale nie ma znaczenia semantycznego)

  • atrybut id - identyfikuje element (przydaje się do nawigacji w obrębie dokumentu)

  • "a href="link">opis łącza a>" - a - linki / łącza wewnątrz dokumentu

  • "element id="moj-identyfikator" element" - atrybut id - identyfikuje element (przydaje się do
  • nawigacji w obrębie dokumentu)

  • "nav" - nawigacja

  • "img src="" alt="" " - img - obrazek o podanym adresie

  • "ul" - unordered list

  • "li" - list item

  • "ol" - ordered list

  • "dl" - description list

  • "dt" - description term

  • "dd" - description details

  • "table" - tabela

  • "tr" - wiersz tabeli (table row)

  • "td" - komórka tabeli (table data (cell))

  • "thead" - nagłówek tabeli

  • "th" - komórka w nagłówku tabeli

  • "tbody" - "ciało" tabeli

  • "caption" - opis tabeli
